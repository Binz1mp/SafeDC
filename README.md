# SafeDC
디시에서 이미지 테러하는 나쁜 친구들을 막기 위해 만든 확장프로그램.

위 아이콘을 클릭하면 크롬 웹 스토어로 이동합니다.


게시글의 이미지를 자동으로 숨김처리해줍니다.


그와 동시에 어떤 버튼을 누르면 다시 이미지가 보입니다.
